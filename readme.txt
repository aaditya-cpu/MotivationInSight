This plugin displays a motivational quote in the WordPress admin bar.

== Description ==

This plugin displays a motivational quote of the day in the WordPress admin bar. It uses an API to fetch the quote and display it on the top bar for the logged-in users. The quotes are in English language.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/motivational-quotes` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. The quotes will be displayed in the admin bar

== Frequently Asked Questions ==

Q: Where do the quotes come from?
A: The quotes are fetched from https://quotes.rest/qod?language=en

Q: Can I change the language of the quotes?
A: Currently, the quotes are only available in English.

Q: Can I change the appearance of the quote in the admin bar?
A: The plugin uses the default WordPress admin bar styling, but you can customize it with CSS.
